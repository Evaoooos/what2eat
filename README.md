# 今天吃什么 (What to Eat Today)

一个基于 AI 的智能菜谱推荐系统，帮助你解决"今天吃什么"的困扰。

## 功能特点

- 🤖 AI 智能推荐菜谱
- 🥘 支持中餐/西餐选择
- ⏰ 按烹饪时间筛选
- 🍜 个性化口味偏好
- 🍱 特殊需求标签（适合带饭/一锅完成/简单厨具）
- 📱 响应式设计，支持移动端

## 技术栈

- Node.js
- Express
- OpenAI API
- HTML/CSS/JavaScript

## 快速开始

1. 克隆项目
```bash
git clone [你的仓库地址]
cd [项目目录]
```

2. 安装依赖
```bash
npm install
```

3. 配置环境变量
创建 `.env` 文件并添加：
```
OPENAI_API_KEY=你的OpenAI API密钥
PORT=3000
```

4. 启动服务器
```bash
npm start
```

5. 访问网站
打开浏览器访问 `http://localhost:3000`
